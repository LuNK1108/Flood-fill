# Flood Fill em Java 2D Graphics

Projeto desenvolvido em Java utilizando **Java 2D Graphics** para aplicar o algoritmo **Flood Fill**, responsável por preencher áreas conectadas de uma imagem a partir de um pixel escolhido.

## Descrição

O sistema permite preencher pixels ao redor de uma área selecionada em uma imagem, utilizando diferentes estruturas de dados para demonstrar o funcionamento do algoritmo:

- Fila
- Pilha
- Nós

O objetivo do projeto é praticar estruturas de dados e manipulação gráfica em Java.

## Funcionalidades

- Carregamento de imagem
- Seleção de ponto inicial para preenchimento
- Preenchimento de pixels conectados
- Implementação com Fila
- Implementação com Pilha
- Utilização de nós para representar posições da imagem
- Interface gráfica com Java 2D Graphics

## Tecnologias utilizadas

- Java
- Java Swing
- Java 2D Graphics
- Estruturas de Dados

## Conceitos aplicados

- Flood Fill
- Pilha
- Fila
- Nós
- Manipulação de pixels
- Recursão/percorrimento iterativo
- Programação orientada a objetos
